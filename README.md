# SSM-Lab

这是一个基于 **SSM（Spring + Spring MVC + MyBatis）** 的 Java Web 练习项目，使用 **Maven 多模块** 方式组织。

## 项目结构

- `SSM000_RegisterAdmin`：管理员注册模块
- `SSM001_LoginAdmin`：管理员登录模块
- `SSM002_LoginInterceptor`：登录拦截器模块
- `SSM003_ManagerAdmin`：管理员管理模块

## 开发环境

- JDK 8
- Maven
- Tomcat 9
- MySQL
- IntelliJ IDEA

## 主要技术

- Spring
- Spring MVC
- MyBatis
- JSP
- JSTL
- Maven
- Tomcat
- MySQL

## 运行说明

1. 使用 IDEA 打开最外层工程
2. 刷新 Maven 依赖
3. 配置 JDK、Tomcat
4. 创建并导入 MySQL 数据库
5. 运行对应模块进行测试

## 说明

本项目主要用于 SSM 框架学习与基础功能练习，包括注册、登录、拦截器和后台管理等内容。
