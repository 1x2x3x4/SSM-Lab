# SSM-Lab

这是一个基于 **SSM（Spring + Spring MVC + MyBatis）** 的 Java Web 练习项目，使用 **Maven 多模块** 方式组织。

## 开发环境

- JDK 8
- Maven
- Tomcat 9
- MySQL
- IntelliJ IDEA

## 运行说明

1. 使用 IDEA 打开最外层工程
2. 刷新 Maven 依赖
3. 配置 JDK、Tomcat
4. 创建并导入 MySQL 数据库
5. 运行对应模块进行测试

## 说明

本项目主要用于 SSM 框架学习与基础功能练习，包括注册、登录、拦截器和后台管理等内容。
